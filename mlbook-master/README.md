## データ
このGitHubに本で使うデータがある。

https://github.com/yosukekatada/mlbook

と、言っても[Kaggle](https://www.kaggle.com/)のデータ等を利用している。

## 環境設定

```
$ python3 -m venv env
$ . env/bin/activate
(env)$ pip install numpy scipy jupyter matplotlib pandas
(env)$ pip install scikit-learn
(env)$ pip install lda
```

## Jupyter Notebookの起動

```
$ jupyter notebook
```
